## Plan Summary
Build a basic arithmetic calculator website as a portfolio piece. Glassmorphism design, vanilla HTML/CSS/JS, keyboard support, responsive layout. No framework, no backend. Single-page static site.

Source brainstorm: `docs/brainstorms/2026-02-26-calculator-website-brainstorm.md`

## Decisions
- **Tech stack:** Vanilla HTML + CSS + JS (no framework, no build step)
- **Design:** Glassmorphism — frosted glass card, backdrop-filter blur, semi-transparent buttons, soft shadows
- **Features:** Basic arithmetic (+, -, *, /), clear, percentage, sign toggle, decimal, keyboard input
- **No history panel**, no themes, no persistence
- **Deployment:** Static files — GitHub Pages compatible
- **Branch strategy:** feat/calculator-website
- **Testing:** Manual browser testing + basic JS unit tests if time allows

## Execution Notes
- This is a portfolio piece — visual polish matters more than feature count
- Keyboard mapping: 0-9, +, -, *, /, Enter (=), Escape (clear), Backspace (delete last), . (decimal)
- Responsive: CSS Grid for button layout, works on mobile and desktop
- Files: index.html, style.css, script.js — that's the entire deliverable
