# Phased Roadmap

## MVP (Minimum Viable Product) — v0.1
**Goal:** Prove the core calculator concept with a polished, responsive web experience using modern design.

### Features
- [ ] Basic arithmetic operations (add, subtract, multiply, divide)
- [ ] Glassmorphism design aesthetic
- [ ] Keyboard support for input and operations
- [ ] Responsive layout (mobile, tablet, desktop)

### Success Criteria
- All arithmetic operations work correctly
- Lighthouse performance score ≥ 95
- Fully functional on desktop and mobile
- Clean, modern visual design implemented

---

## V1 (First Full Release) — v1.0
**Goal:** Transform the calculator into a complete, publishable product with history tracking and polish.

### Features (in addition to MVP)
- [ ] Calculation history sidebar
- [ ] Theme toggle (light/dark mode)
- [ ] Improved animations and transitions
- [ ] Shell completions for developers

### Success Criteria
- Published to GitHub Pages
- README with working demo GIF
- All V0 features stable and documented
- Version bump to v1.0

---

## V2 (Enhancement Release) — v2.0
**Goal:** Expand capabilities with advanced features and offline support for a professional tool.

### Features (in addition to V1)
- [ ] Scientific calculator mode toggle
- [ ] localStorage persistence for history and settings
- [ ] PWA (Progressive Web App) support
- [ ] Accessibility audit and compliance

### Success Criteria
- 100+ GitHub stars
- Full offline functionality via PWA
- Scientific operations working correctly
- WCAG 2.1 AA accessibility standards met

---

## Future / Backlog
- Multi-calculator hub (grouped calculators for different domains)
- Sharing calculations via URL encoding
- Embedded widget mode for other websites
